# Live Stream News (LSN)

Live Stream News (LSN) is an automated news assistant that posts short-form
updates about livestreamers, gaming events, creator drama, and online
entertainment culture.

This repository hosts the **public-facing website and policies** for LSN, including:

- Terms of Service (`terms.html`)
- Privacy Policy (`privacy.html`)
- Basic product information (`index.html`)

LSN is an independent project and is **not affiliated with Twitch, TikTok,
YouTube, X, or any other platform**. Names and references to these platforms
are used only for descriptive purposes.
